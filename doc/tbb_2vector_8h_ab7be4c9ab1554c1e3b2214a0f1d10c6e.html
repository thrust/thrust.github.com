tbb::universal_vector is a container that supports random access to elements, constant time removal of elements at the end, and linear time insertion and removal of elements at the beginning or in the middle. The number of elements in a tbb::universal_vector may vary dynamically; memory management is automatic. The elements contained in a tbb::universal_vector reside in memory accessible by the tbb system and host systems.
