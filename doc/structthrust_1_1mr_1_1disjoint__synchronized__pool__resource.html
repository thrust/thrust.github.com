Detailed Description

template<typename Upstream, typename Bookkeeper>
struct thrust::mr::disjoint_synchronized_pool_resource< Upstream, Bookkeeper >

A mutex-synchronized version of disjoint_unsynchronized_pool_resource. Uses std::mutex, and therefore requires C++11.

Template Parameters
Upstreamthe type of memory resources that will be used for allocating memory blocks to be handed off to the user
Bookkeeperthe type of memory resources that will be used for allocating bookkeeping memory
