pointer< T > thrust::system::omp::malloc ( std::size_t  n)

Allocates an area of memory available to Thrust's omp system.

Parameters
nNumber of bytes to allocate.
Returns
A omp::pointer<void> pointing to the beginning of the newly allocated memory. A null omp::pointer<void> is returned if an error occurs.
Note
The omp::pointer<void> returned by this function must be deallocated with omp::free.
